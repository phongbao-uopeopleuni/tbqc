# Folder SQL

Folder này chứa các file SQL phụ trợ và migration.

## File quan trọng ở root:
- `database_schema.sql` - Schema chính
- `database_schema_extended.sql` - Schema mở rộng
- `database_schema_in_laws.sql` - Schema quan hệ con dâu/con rể
- `database_schema_final.sql` - Schema final
- `migration_manual.sql` - Migration thủ công
- `migration_add_fm_id.sql` - Migration thêm FM_ID
- `check_and_migrate.sql` - Kiểm tra và migrate
- `check_database_status.sql` - Kiểm tra trạng thái database

## File trong folder này:
Các file SQL phụ trợ, setup, và migration bổ sung.
