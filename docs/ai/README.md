# AI Docs

This folder isolates material intended for AI-assisted workflows.

## Structure

- `agents/`: agent policies, skills, and capability references
- `memory/`: quick-start context for AI sessions
- `reports/`: AI-generated summaries worth retaining
- `logs/`: work logs for general AI-assisted sessions (non-workstream tasks)
- `antigravity/`: prompts, onboarding, and work logs for the Antigravity security workstream

## Notes

- These files are useful for repeatable AI collaboration.
- They are not the primary source of truth for production behavior.
- Canonical maintainer context lives in `docs/operations/system-context.md`.
- Do not mix operational runbooks or product requirements into this area.
