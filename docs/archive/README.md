# Archived Docs

This folder contains older planning notes and maintenance history that are still worth keeping for traceability.

## Structure

- `pre-refactor/`: historical planning snapshots and prep notes
- `maintenance/`: retired cleanup or maintenance notes

## Rules

1. Archived documents are read-only unless there is a strong reason to annotate them.
2. Do not treat archive files as the current source of truth.
3. When replacing a canonical document, move the old one here instead of leaving duplicates at the root of `docs/`.
