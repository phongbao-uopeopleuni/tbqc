# TBQC

TBQC is a Flask + MySQL web application for family genealogy, member access, admin operations, activities, gallery management, and grave-site information.

## Documentation

- Docs index: `docs/README.md`
- System context: `docs/operations/system-context.md`
- Product requirements: `docs/product/srs.md`
- Operations runbook: `docs/operations/runbook.md`
- Maintenance guide: `docs/operations/maintenance.md`
- Incident and hotfix log: `docs/operations/incident-log.md`
- Security baseline: `docs/security/security.md`
- Release history: `docs/releases/changelog.md`

## Development

- App entrypoint: `app.py`
- Production runtime: `Procfile`
- Render fallback config: `render.yaml`
- Tests: `pytest`
- JS lint: `npm run lint`
