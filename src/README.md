# Thư mục `src` (frontend / module JS)

Đặt mã nguồn cần **hiển thị trên Knowledge Graph** (Admin Dashboard) vào đây.

Sinh file đồ thị:

```bash
cd scripts/code-graph
npm install
npm run scan
```

Hoặc quét thư mục khác (ví dụ `static/js`):

```bash
npm run scan:static
```

Đầu ra: `static/data/code-graph.json`.
